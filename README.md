# GreenSock Animation Platform (GSAP)

Exploring GreenSock Animation Platform (GSAP) with practical examples, creative animations, and real-world UI interactions.

## Overview

This repository demonstrates the power and flexibility of GSAP (GreenSock Animation Platform) through a series of progressive tasks, from basic animations to complex, real-world UI interactions.

## What is GSAP?

GSAP is a robust JavaScript animation library that provides a performant, lightweight solution for creating smooth, professional animations in web applications. It's widely used in production environments by top companies and developers worldwide.

## Repository Structure

The repository is organized into task-based folders, each building upon previous knowledge:

### Tasks

- **Task-00**: Foundation and setup
- **Task-01**: Basic animation concepts
- **Task-02**: Card animations with stagger effects
- **Task-03**: Advanced animation techniques
- **Task-04**: Hero section animation

Each task folder contains:
- `index.html` - HTML structure and animation implementation
- `image.png` - Visual reference or assets
- Supporting animation scripts

## Best Practices

When using GSAP:
- Use `gsap.to()` for animating to a final state
- Use `gsap.from()` for animating from an initial state
- Use `gsap.fromTo()` for precise control over start and end states
- Organize animations into timelines for complex sequences
- Consider performance when animating multiple elements
- Use appropriate easing functions for natural motion

## Performance Considerations

- GSAP is highly optimized for performance
- Use GPU-accelerated properties (transform, opacity)
- Avoid animating properties that trigger layout recalculations
- Test animations on various devices
- Consider using `will-change` CSS property strategically
